Integer division Open on JetBrains Academy

You need to write a program that reads two values and prints the integer division. To divide, there is a special function called intDivision.

In the input, it accepts two values of the type String and returns a value of the type Int. In case of successful conversion to Int, the function intDivision returns the result of dividing the first input parameter by the second. If the division fails, the default value 0 (zero) should be returned.

If an arithmetic exception occurs when dividing by 0, it shall return the message Exception: division by zero!. If it cannot convert the input parameters to the Int type, print Read values are not integers.

Use the try/catch statement.


Sample Input:
16
2
Sample Output:
8

Sample Input:
5
0
Sample Output:
Exception: division by zero!
0

Sample Input:
Bumblebee
24
Sample Output:
Read values are not integers.
0
