The function printFifthCharacter represented below prints the 5th character of the input string. However, it does not allow the possibility to have less than 5 characters in the entered word. In such cases, the StringIndexOutOfBoundsException will be thrown.

Write code that in the case of such an exception will print "The input word is too short!" and will not interrupt the execution of the program.

Use the try/catch statement.


Sample Input:
Bumblebee

Sample Output:
The fifth character of the entered word is l

Sample Input:
155

Sample Output:
The input word is too short!

Sample Input:
Kotlin

Sample Output:
The fifth character of the entered word is i
